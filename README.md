# 실행

터미널에서 원하는 폴더에 들어간뒤

git clone https://github.com/junhee-won-ringle/selenium
  -> 파일이 받아짐

npm install
  -> 프로젝트 실행에 필요한 파일들 설치됨

node app.js
 -> 테스트 실행
