# 실행

터미널에서 원하는 폴더에 들어간뒤,
<br/>
<br/>
git clone https://github.com/junhee-won-ringle/selenium
<br/>
-> 파일이 받아짐

<br/>
<br/>
npm install
<br/>
  -> 프로젝트 실행에 필요한 파일들 설치됨

<br/>
<br/>
.env 파일 생성

<br/>
<br/>
.env 파일에 내용 입력
<br/>
QA1_PAGE="https://QA1주소"

QA2_PAGE="https://QA2주소"

QA_ID="QA 아이디"

QA_PW="QA 비밀번호"

QA_FIRST_ID="QA 처음 아이디"

QA_FIRST_PW="QA 처음 비밀번호"

<br/>
<br/>
node app.js
<br/>
 -> 테스트 실행
